# Einkaufsliste App

Moderne, elegante Android Einkaufslisten-App mit Live-Synchronisation via Firebase.

## Features
- Dark Mode Theme (edles Design)
- Echtzeit-Sync zwischen mehreren Nutzern
- Firebase Auth & Firestore
- Jetpack Compose UI

## Setup
1. Erstelle Firebase Projekt und lade google-services.json herunter.
2. Ersetze in app/google-services.json.
3. Baue mit Android Studio oder `./gradlew build`.

## GitHub Workflow
